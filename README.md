# PSP
AKTUALIZACJA SYSTEMU NA WERJSĘ 6.61

POBIERZ REPOZYTIRUM NA PC, KLIKNIJ ZIELONY PRZYCIK CODE I WYBIERZ POBIERZ ZIP

**Przed przystąpieniem do wgrywania oprogramowania PSP musi widzieć kartę pamięci**
Jeśli jej nie widzi:
1.Sformatuj kartę do formatu FAT32 (na PC - skorzystaj z guiformat.exe)
2.Włóż kartę do PSP i w ustawieniach sformatuj kartę ponownie
3.Teraz karta powinna działać i posiadać domyślne katalogi

**Instalacja oprogramowania ARC**
1. Pobierz repozytorium na PC (format zip)
2. Wypakuj Repo
3. Odszukaj PSP\\SAVEDATA
4. Skopiuj na swoje PSP w lokalizacji PSP\SAVEDATA folder ARK\_01234
5. Odszukaj lokalizacji PSP\GAME
6. Skopiuj na swoje PSP w lokalizacji PSP\GAME folder ARC_Loader
7. Włóż kartę do PSP i uruchom ARC (Loader jest tam gdzie gry)
8. Gdy konsola pomyslnie się zaaktualizuje i uruchomi ponownie sformatuj ją ponownie w PSP
9. Podłącz ją do PC i sprawdź czy w karcie utworzył się folder ISO
11. Przejdź do folderu ISO i wgraj grę (Link do gier poniżej)
12. Możesz też przenieś foldery z grami Minecraft i Spelunky do PSP\\GAME
13. Teraz gry z karty pamięci powinny działać
14. Jeśli wyjmiesz baterie konsola usunie oprogramowanie ARC, dlatego powtórz kroki 3-6 aby mieć zapisany Loader ARC w PSP
15. Jeśli konsola nie będzie chciała uruchomić gry i wyskoczy błąd to odpal jeszcze raz ARC Loader w PSP
    
**Gry możesz pobierać z https://myrient.erista.me/files/Redump/Sony%20-%20PlayStation%20Portable/**

**INSTALACJA OFICJALNEGO SYSTEMU 6.61**
Pobierz repozytorium na PC (format zip)
Podłącz PSP lub kartę do komputera.
Wejdź na kartę pamięci do folderu PSP, potem GAME. Skopiuj tam z pobramego zipa folder o nazwie UPDATE.
Wklej tam pobrany plik. Ścieżka powinna wyglądać tak: Karta:/PSP/GAME/UPDATE/EBOOT.PBP.
Odłącz konsolę, wejdź w menu Game -> Memory Stick i uruchom aktualizację.
Uwaga: Bateria musi mieć min. 50% i najlepiej, żeby konsola była pod ładowarką.
